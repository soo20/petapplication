# Yuna Pet Application Project


## Project Overview

This mobile application project addresses the lack of accessible and detailed information about various pet species and their specific needs. It aims to improve pet care practices through a user-friendly interface and real-time updates using Flutter and Firebase.

## Objectives

1. **Pet Species Information:** Provide detailed and accessible information about various pet species, including their dietary needs, exercise requirements, and general care tips.
2. **Skin Disease Detection in Dogs:** Develop a feature to identify skin diseases in dogs, helping pet owners recognize signs early for prompt treatment.
3. **Pet Care Reminders:** Implement features to remind pet owners of their responsibilities, ensuring no aspect of pet care is overlooked due to busy schedules.

## Technologies Used

- **Flutter:** For building the cross-platform mobile application.
- **Firebase Firestore:** For a real-time database to store pet species information and user data.
- **Firebase Authentication:** For secure user authentication and authorization.
- **Machine Learning (ML) Integration:** Integrate a skin disease detection model (if applicable) using Firebase ML Kit.
- **Google Cloud Functions:** For backend logic and integration with Firebase services.
- **Flutter Packages:** Utilize various Flutter packages for UI components, state management, and Firebase integration.

## Features

1. **Pet Species Profiles:** Detailed profiles for various pet species, including dietary guidelines, exercise routines, and healthcare recommendations.
2. **Skin Disease Identification:** A feature to detect common skin diseases in dogs based on symptoms.
3. **Pet Care Scheduler:** Reminders and notifications for feeding times, vet appointments, and grooming sessions to support busy pet owners.

## Implementation Steps

1. **App Design and Development:**
   - Designing a user-friendly interface using Flutter widgets and Firebase integration.
   - Developing frontend functionalities for browsing pet species profiles and entering symptoms for disease detection.

2. **Firebase Integration:**
   - Setting up Firebase Firestore for storing pet care information and user data securely.
   - Implementing Firebase Authentication for user login and registration.

3. **Feature Development:**
   - Integrating machine learning models (if applicable) for skin disease detection using Firebase ML Kit.
   - Implementing pet care scheduler functionalities with real-time notifications using Firebase Cloud Messaging.

4. **Testing and Deployment:**
   - Conducting thorough testing to ensure the application's functionality, performance, and user experience.
   - Deploying the application on Android and iOS platforms through the Google Play Store and Apple App Store.

5. **Documentation:**
   - Documenting the project, including setup instructions, features implemented, and future enhancements.

## Expected Outcomes

- A user-friendly mobile application providing comprehensive pet care information and management tools.
- Improved awareness and understanding of pet health through accessible resources and real-time disease detection capabilities.
- Enhanced pet care management for busy pet owners, promoting timely and organized care routines.

## Conclusion

This mobile application project leverages Flutter and Firebase to empower pet owners with accessible information and tools for effective pet care management. By combining technology with pet health expertise, we aim to enhance the well-being and happiness of pets and their owners.
